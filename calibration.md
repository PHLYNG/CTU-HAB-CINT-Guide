# Calibration - How Do We Know the Measurements Are Correct?

In order to have reliable data, we need to test our components on the ground to make sure the data they are outputting is correct.

To calibrate our sensors, we will need to use breadboards to begin prototyping.

Begin by connecting each sensor individually to the ESP32. Each sesnor will have a pinout diagram that explains how wires need to be connected so that the sensor receives power (and how much), data is recorded and transferred, and that the sensor is properly grounded.
